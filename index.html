Visualize your code architecture instantly. Convert any repository into an interactive dependency graph — no installation, no server, no account required. Supports 12 programming languages, up to 250 source files.

¿Qué es GraphMyCode?

GraphMyCode convierte cualquier repositorio de código en un grafo interactivo de dependencias que se ejecuta completamente en tu navegador. Sin instalación, sin servidor, sin cuenta. Analiza hasta 250 archivos fuente de 12 lenguajes de programación. Todo el procesamiento ocurre mediante WebAssembly (tree-sitter, HuggingFace transformers) directamente en tu máquina.

What is GraphMyCode?

GraphMyCode converts any code repository into an interactive dependency graph running entirely in your browser. No installation, no server, no account required. Analyzes up to 250 source files across 12 programming languages using WebAssembly (tree-sitter, HuggingFace transformers).

¿Cómo funciona?

Sube un .zip de tu proyecto o pega una URL de GitHub público. GraphMyCode parsea el código con tree-sitter (WebAssembly), construye un grafo de símbolos y dependencias en memoria y lo renderiza de forma interactiva. Tu código nunca abandona el navegador.

How does it work?

Upload a .zip of your project or paste a public GitHub URL. GraphMyCode parses your code with tree-sitter (WebAssembly), builds a symbol and dependency graph in memory, and renders it interactively. Your code never leaves your browser.

Cuatro vistas para entender cualquier código

Vista Structural — Grafo de dependencias interactivo

Grafo 2D interactivo de archivos, clases, funciones e importaciones. Responde: ¿qué importa este fichero? ¿quién llama a esta función? ¿qué módulos están aislados? ¿cómo fluye la ejecución?

Vista Semantic 3D — Similaridad semántica del código

Agrupa nodos por similaridad de código usando embeddings locales (HuggingFace WASM) + UMAP + algoritmo de Leiden. Detecta lógica duplicada y analiza el impacto real de un cambio más allá de las dependencias directas.

Vista Technical Debt City — Deuda técnica como ciudad 3D

Visualiza el repositorio como una ciudad 3D. El edificio más alto de cada barrio tiene más deuda técnica (mayor acoplamiento). Identifica los ficheros más difíciles de cambiar y el código más acoplado.

Vista Dependency Heatmap — Mapa de calor de dependencias

Muestra el acoplamiento real entre archivos. Detecta ciclos de importación y código espagueti. Las dependencias bidireccionales aparecen como aristas naranjas. Identifica exactamente dónde romper dependencias.

Privacidad: sin servidor ni base de datos

Todo corre en tu navegador, en memoria. Al cerrar la pestaña, desaparece sin dejar rastro. Sin telemetría propia, sin datos personales. Zero server requests — all analysis runs in-browser via WebAssembly.

IA para preguntar sobre tu código (opcional)

Conecta tu propia API key (OpenAI, Gemini, Anthropic u Ollama) y hazle preguntas en lenguaje natural sobre tu codebase. Con Ollama, el código no sale de tu máquina.

Lenguajes de programación soportados

JavaScript, TypeScript, Python, Java, Go, Rust, C, C++, C#, PHP, Ruby y Swift. Soporta hasta 250 archivos fuente por repositorio.
