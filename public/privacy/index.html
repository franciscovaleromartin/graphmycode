Privacidad

Política de privacidad

GraphMyCode no recopila, no almacena y no transmite ningún dato personal ni ninguna línea de tu código. Esta página explica exactamente qué ocurre con tu información cuando usas la herramienta.

Resumen en una línea: tu código nunca sale de tu navegador. Nada llega a ningún servidor nuestro porque no existe ningún servidor nuestro.

Análisis de código — 100% en tu navegador

GraphMyCode funciona íntegramente en el cliente. El análisis de código se realiza mediante WebAssembly (WASM) directamente en tu navegador:

  • tree-sitter parsea el código fuente sin enviar ningún byte a ningún servidor.
  • El grafo de dependencias se construye en memoria RAM del navegador.
  • Los embeddings semánticos (vista Semantic 3D) se calculan con @huggingface/transformers vía WASM — también localmente.
  • Al cerrar la pestaña, toda la información desaparece sin dejar rastro.

No existe ninguna base de datos, ningún backend propio ni ningún almacenamiento persistente asociado a GraphMyCode.com.

Carga de repositorios de GitHub

Cuando usas la opción "URL de GitHub", GraphMyCode llama directamente a la API pública de GitHub desde tu navegador:

  • Solo funciona con repositorios públicos.
  • Los archivos se descargan directamente de raw.githubusercontent.com a tu navegador.
  • La petición proviene de tu IP, no de nuestros servidores.
  • GitHub puede registrar estas peticiones según su propia política de privacidad.

Función de IA (opcional)

La función de chat con IA es completamente opcional y está desactivada por defecto. Si decides usarla:

  • Debes proporcionar tu propia API key (OpenAI, Google Gemini, Anthropic o Ollama).
  • La API key se guarda únicamente en el sessionStorage de tu navegador y se borra al cerrar la pestaña.
  • Cuando haces una pregunta, fragmentos del grafo de código se envían al proveedor de IA elegido. La cantidad de código enviada depende del contexto de la pregunta.
  • Cada proveedor tiene sus propias condiciones de uso y política de privacidad. GraphMyCode no tiene acceso ni control sobre el tratamiento que el proveedor hace de esa información.

Con Ollama, el modelo corre localmente en tu máquina y ningún dato sale a internet.

Analítica web (Vercel Analytics)

GraphMyCode usa Vercel Analytics para recopilar métricas de uso anónimas:

  • Páginas visitadas y eventos básicos de interacción (p. ej. "grafo visualizado").
  • No incluye datos personales identificables ni contenido de código.
  • Gestionado por Vercel según su política de privacidad.

Cookies

GraphMyCode no usa cookies propias. Vercel Analytics puede usar cookies de sesión técnicas mínimas para contabilizar visitas únicas.
